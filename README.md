# SRT/KTX 예매 프로그램

매진된 기차표 예매를 도와주는 파이썬 프로그램입니다.  
원하는 표가 나올 때 까지 새로고침하여 예약을 시도합니다.

## 다운
[릴리즈](https://github.com/dhgwag/train_reservation/releases/latest )에서 최신 파일 다운로드  
Windows : 기차표예매.exe  
Mac OS(OS X) : 기차표예매.app
